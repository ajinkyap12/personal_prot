# Ajinkya Patil - Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and experience as a Full Stack Developer.

## 🚀 Features

- **Modern Design**: Clean and professional UI with glassmorphism effects
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Interactive animations and scroll effects
- **Skills Showcase**: Animated skill cards with progress indicators
- **Project Gallery**: Modern project cards with hover effects
- **Contact Form**: Easy-to-use contact section

## 🛠️ Technologies Used

- HTML5
- CSS3 (with Glassmorphism effects)
- JavaScript (ES6+)
- Bootstrap 4
- Particles.js
- AOS (Animate On Scroll)
- jQuery

## 📦 Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio.git
```

2. Open `index.html` in your web browser

3. (Optional) For local development with a server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```
